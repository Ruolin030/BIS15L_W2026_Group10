#Background:
Gestation length is an important life-history trait. Life-history traits are characteristics that describe how organisms grow, survive, and reproduce. 
These traits include lifespan, number of offspring, gestation length, and the age at which offspring are weaned. 
Together, these traits influence an organism’s overall reproductive success and survival.
